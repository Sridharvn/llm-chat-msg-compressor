# Security Policy

## Supported Versions

We currently support the following versions of `llm-chat-msg-compressor`:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you believe you have found a security vulnerability, please report it to us as follows:

1.  **Do not open a public issue.**
2.  Email the details of the vulnerability to sridhar.v.nampoothiripad@gmail.com.
3.  We will acknowledge your report within 48 hours.
4.  We will provide a timeline for fixing the issue and keep you updated on our progress.
5.  Once the issue is resolved, we will post a security advisory and credit you for the discovery (if desired).

Please include as much information as possible, including:

- Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting, etc.)
- Location of the issue (e.g., file name, function name, etc.)
- Steps to reproduce the issue
- Potential impact of the issue

Thank you for helping us keep `llm-chat-msg-compressor` secure!
